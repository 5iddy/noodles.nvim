# Neovim Config


## Requirements

- `NVIM v0.8.2`

Its the latest version of archlinux that this repo will be synced to.

## Installation

Clone this repo as your `~/.config/nvim`

```bash
git clone https://github.com/5iddy/noodles.nvim ~/.config/nvim
```
